# 📅 Event Calendar App (Next.js)

A responsive and interactive calendar application built using Next.js. This app allows users to navigate between months, identify the current date, and manage notes for specific days with a clean and intuitive UI.

## 🚀 Features

- 📍 **Today Highlight** — Clearly indicates the current date  
- 📆 **Month Navigation** — Seamless switching between months  
- 📝 **Notes per Day** — Add and view notes for selected dates  
- 🎨 **Responsive UI** — Works across desktop and mobile screens  
- ⚡ **Fast & Optimized** — Built with Next.js for performance  

---

## 🛠 Tech Stack

- **Frontend:** Next.js, React.js  
- **Styling:** Tailwind CSS  
- **State Management:** React Hooks

🧠 Key Implementation Details
Dynamic calendar grid generation based on month and year
Conditional rendering for highlighting today’s date
State-driven UI updates for selected dates and notes
